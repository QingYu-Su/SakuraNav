# Development Guide

完整的开发指南请参考[中文版开发指南](/guide/dev-guide)。

## Quick Start

```bash
npm install
cp config.example.yml config.yml
npm run dev
```

## Build & Deploy

```bash
npm run build:start           # Production build and start
npm run build:start:skip-lint # Skip lint
npm run build:start:skip-build # Skip build
```
