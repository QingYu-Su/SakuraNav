# 目录结构

完整的项目目录结构请参考[中文版目录结构](/guide/directory-structure)。

核心目录概览：

| 目录 | 说明 |
|:-----|:-----|
| `src/app/` | Next.js App Router（页面 + API 路由） |
| `src/components/` | React 组件（sakura-nav/admin/auth/dialogs/ui） |
| `src/lib/` | 工具库（base/config/database/utils/services） |
| `src/hooks/` | 自定义 React Hooks |
| `storage/` | 数据存储（运行后生成） |
