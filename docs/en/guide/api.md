# API Reference

完整的 API 接口文档请参考[中文版 API 接口](/guide/api)。

接口分组概览：

| 分组 | 认证要求 | 说明 |
|:-----|:---------|:-----|
| 认证接口 | — | 登录/登出/注册/OAuth |
| 导航接口 | — | 公开网站/标签/笔记列表 |
| 管理接口 | 需认证 | CRUD/排序/在线检测 |
| 管理员接口 | 管理员 | 引导数据/用户管理/OAuth配置 |
| 配置接口 | 管理员 | 导入/导出/重置 |
| 用户数据接口 | 需认证 | 按用户隔离的导入/导出/重置 |
| AI 接口 | 需认证 | 推荐/分析/检查 |
| 社交卡片接口 | 需认证 | CRUD/排序 |
| 笔记卡片接口 | 需认证 | CRUD/附件管理 |
| 快照接口 | 需认证 | 创建/恢复/删除/重命名 |
